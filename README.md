# Bundle program
[![Build Status](https://travis-ci.org/dawidd6/bundle.svg?branch=master)](https://travis-ci.org/dawidd6/bundle)
### Installing from source 
```sh
git clone https://github.com/dawidd6/bundle.git
cd bundle
make
sudo make install
```
### Usage
To print help message, type
```sh
bundle
#or
bundle -h
#or
bundle --help
```
